# How to Unlock the Portfolio Source Code ?

To get instant access to the private assignment vault, complete these three simple steps:

**Step 1: Drop a Star**
Scroll to the top right of this page and click the **Star** (⭐) button on this repository.

**Step 2: Authenticate**
Click the secure link below to verify your star. It will ask you to log in with your GitHub account so the system can confirm you've starred the repo.
 **[Unlock the Source Code Here](https://portfolio-sourcecode.streamlit.app/)**

**Step 3: Accept your Invite**
Once verified, the system will automatically invite you to the private `SatyamChandraPortfolio` repository as a collaborator. Check your email or your GitHub notifications to accept the invite!
